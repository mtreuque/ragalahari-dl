## ⚙️ How to Use ragalahari-dl

The program works through simple commands you type or paste.

### Step 1: Copy Gallery URL

Visit Ragalahari and open the gallery page you want to download images from.

Copy the full web address (URL) from your browser’s address bar.

### Step 2: Use the Command Window

In the ragalahari-dl window, type or paste this command format:

```
python ragalahari-dl.py "gallery-url"
```

Replace `"gallery-url"` with the actual link you copied.

For example:

```
python ragalahari-dl.py "https://www.ragalahari.com/gallery1"
```

If you do not have Python installed, you may see instructions below on how to install it or use the given executable that works without Python.

### Step 3: Start the Download

Press Enter after typing the command.

The program will begin downloading all images from that gallery.

You will see the progress in the window, including the number of files saved.

Images will save by default in a new folder inside your “Documents” folder named after the gallery or date.

### Step 4: Find Your Images

When the download finishes, open the folder created during the process.

All the photos will be there, ready for viewing or sharing.
